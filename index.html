Installation

1. Clone the repository

{% highlight shell %} git clone https://github.com/achillesrasquinha/Minitron.js.git {% endhighlight %}

2. Install using npm

{% highlight shell %} npm install minitron.js {% endhighlight %}

3. Install using bower

{% highlight shell %} bower install minitron.js {% endhighlight %}

In your HTML file

{% highlight html %} ... ... {% endhighlight %}

Usage

A template declaration would be as follows

{% highlight javascript %} Minitron.make("", { /**/ }); {% endhighlight %}

Methods

Return Method Description
Minitron setMessage(message) Initializes/Updates the message to-be displayed.
Minitron setOptions(options) Pass your options to settings to change them any time.

Options

{% highlight javascript %} Minitron.make("", { color: '#00FFFF' /*default*/ | "", speed: 50 /*default*/ | /* higher the speed value, slower to render*/ }); {% endhighlight %}
